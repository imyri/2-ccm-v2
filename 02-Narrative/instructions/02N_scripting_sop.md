# Scripting SOP (02N_scripting_sop.md)

## Objective
Convert technical research into high-retention, viral narratives using the "Pragmatic Architect" persona.

## The Scripting Framework
1. **The Hook (0-3s)**: Challenge a common belief or show a shocking result.
2. **The Problem (3-10s)**: Agitate the technical friction our target persona feels.
3. **The Solution (The Meat)**: Brief, high-value technical walk-through.
4. **The CTA (Call to Action)**: Direct and value-based.

## Tone/Voice
- **Authority**: High. Use specific terms (e.g., "MCP", "Context Window").
- **Accessibility**: Simple sentences. No "fluff".
- **Aesthetic**: Minimalist and sleek.

## Platform Rules
- **YouTube Shorts**: Fast cuts, text overlays mentioned in brackets.
- **X Threads**: 1. High-impact hook 2. The "What" 3. The "How" 4. The Tool 5. CTA.
