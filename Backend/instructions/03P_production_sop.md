# Production SOP (03P_production_sop.md)

## Objective
Assemble the narrative script and visual assets into a high-authority final video/post.

## The Fabrication Workflow

1. **Shot List Generation**: Run `/motionmuse [script_path]` to generate the `03P_shot_list.json`.
2. **Asset Procurement**: 
   - Check `03-Production/Assets/` for generated images.
   - For B-roll, use free stock sites (Pexels/Pixabay) based on the visual descriptions in the shot list.
3. **Audio Production**:
   - Use free TTS (Text-to-Speech) like **ElevenLabs (Free tier)** or local **Bark/Coqui** if available.
   - Narrative tone must match the "Pragmatic Architect" persona.
4. **Final Assembly (Manual)**:
   - Import assets into an editor (CapCut/Premiere).
   - Apply the **Sleek Dark Mode Tech** aesthetic (Deep blues, high contrast, minimalist transitions).
   - Sync text overlays to the audio cues as defined in the shot list.

## Visual Standards
- **Resolution**: 1080x1920 (9:16) for Shorts/TikTok.
- **Overlays**: Clean, sans-serif fonts (Inter/Outfit). Maximum 5 words per overlay.
