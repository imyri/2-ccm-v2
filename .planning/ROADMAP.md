# Roadmap: CCM — AI-Powered Content Creation Machine

## Overview

From an empty directory to a fully autonomous content production engine. This roadmap covers the construction of the I.A.M. framework, the deployment of signal-gathering modules, the orchestration of agentic narrative generators, and the automated distribution of high-authority content across social platforms.

## Phases

- [ ] **Phase 1: Foundation** - Implement the I.A.M. core architecture and workspace routing.
- [ ] **Phase 2: Signal Intelligence** - Deploy `/trendtracer` and `/specspy` for automated trend research.
- [ ] **Phase 3: Narrative & Fabrication** - Implement `/scriptsmith` and `/motionmuse` for content generation.
- [ ] **Phase 4: Distribution & Growth** - Connect `/echoblast` and `/nexusnode` for omnichannel impact.

---

## Phase Details

### Phase 1: Foundation
**Goal**: Establish the "brains" and "muscles" infrastructure.
**Depends on**: Nothing
**Requirements**: CORE-01, CORE-02, CORE-03, CORE-04
**Success Criteria**:
  1. Obsidian-style Context Rooms (00-06) exist with proper hierarchy.
  2. Local brand instructions (`CONTEXT.md`) are successfully parsed by the Agent layer.
  3. A test Python module can be called and its output routed to a specific workspace.
**Plans**: 2 plans

Plans:
- [ ] 01-01: Setup workspace hierarchy and routing logic.
- [ ] 01-02: Implement Agent-Module bridge (I.A.M. core).

### Phase 2: Signal Intelligence
**Goal**: Automated discovery of viral triggers and deep research.
**Depends on**: Phase 1
**Requirements**: DISC-01, DISC-02
**Success Criteria**:
  1. `/trendtracer` successfully dumps niche-specific signals into `01-Discovery/Trends/`.
  2. `/specspy` generates a structured Markdown report from raw signal data.
**Plans**: 2 plans

Plans:
- [ ] 02-01: Implement Signal Scrapers (YouTube/X).
- [ ] 02-02: Implement Research Agent orchestration.

### Phase 3: Narrative & Fabrication
**Goal**: Transforming research into high-retention multimedia.
**Depends on**: Phase 2
**Requirements**: NARR-01, NARR-02, PROD-01, PROD-02
**Success Criteria**:
  1. `/scriptsmith` produces a multi-platform script JSON based on research reports.
  2. `/motionmuse` renders a draft video with captions and voiceover from the script.
**Plans**: 3 plans

Plans:
- [ ] 03-01: Persona-based script generator development.
- [ ] 03-02: Video template engine integration (Remotion/MoviePy).
- [ ] 03-03: Media asset assembly & voiceover synthesis.

### Phase 4: Distribution & Growth
**Goal**: Closing the loop with platform posting and community delivery.
**Depends on**: Phase 3
**Requirements**: DIST-01, DIST-02, MONY-01, MONY-02
**Success Criteria**:
  1. `/echoblast` publishes content to at least one social API (e.g., X or Meta).
  2. Linktree/Affiliate links are automatically rotated based on the content topic.
**Plans**: 2 plans

Plans:
- [ ] 04-01: Social API integration and scheduler.
- [ ] 04-02: Affiliate/Monetization logic & final loop closure.
