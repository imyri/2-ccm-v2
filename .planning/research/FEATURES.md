# FEATURES.md — Product Surface

## Table Stakes (Must Have)

- **Omnichannel Scripting**: Generate platform-specific versions (Shorts, Threads, FB posts) from a single signal.
- **Programmatic Video**: Dynamic B-roll, captions, and branding based on script JSON.
- **Trend Detection**: Daily scanning of niche-specific signals (YouTube/X).
- **Scheduled Publishing**: Set-and-forget distribution.

## Differentiators (Competitive Edge)

- **I.A.M. Grounding**: Using local `.agent/instructions` to enforce ultra-specific brand personas that don't sound like "AI generic".
- **Feedback-Loop Refinement**: Automatically ingest performance metrics (views/likes) to refine future system prompts.
- **Obsidian-as-UI**: Human-in-the-loop interaction via a local Markdown vault, making the automation audit-able and transparent.
- **Context Room Routing**: Intelligent agent navigation that minimizes token usage by only reading relevant workspace data.

## Anti-Features (Deliberately Not Building)

- **Manual Timeline Editor**: We are building a machine, not a creative suite. If it can't be automated via code/instructions, it doesn't belong here.
- **Paid Ad Management**: Initially focus on organic growth and affiliate revenue.
