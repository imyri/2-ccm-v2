## 🚀 Setup & Installation

### 1. Prerequisites
- **Python 3.10+**
- **n8n** (Optional: for HITL approvals)
- **FFmpeg** (Required for multimedia processing)

### 2. Environment Configuration
Create a `.env` file in the root directory:

```bash
# Core AI
GEMINI_API_KEY=your_gemini_api_key

# n8n Automation (HITL)
HITL_ENABLED=true
N8N_HITL_WEBHOOK_URL=http://your_n8n_webhook_url
```

### 3. Installation
```powershell
# Install dependencies from the Backend folder
pip install -r Backend/requirements.txt

# Start the Control Center
.\START_CCM.bat
```

## 🛠️ Operational Flow

### Phase 1: Discovery & Research
`TrendTracer` and `SpecSpy` identify high-authority potential and generate intelligence reports. Output is saved to `Db/Signals` and `Db/Research`.

### Phase 2: Narrative Generation
`ScriptSmith` transforms signals into viral storytelling. Output is saved to `Db/Scripts`.

### Phase 3: Production Mapping
`MotionMuse` breaks the script into a structured visual shot list. Output is saved to `Db/Media`.

## ⚖️ Governance & Rules (I.A.M. V2)

1. **Instructions (I)**: AI Agents must read instruction files in `Backend/instructions/` before execution.
2. **Agents (A)**: Logic is handled by `gemini-2.0-flash` for high-performance reasoning.
3. **Modules (M)**: All heavy lifting is done by deterministic scripts in `Backend/modules/`.
4. **Context Isolation**: Modules only write to their designated `Db/` subdirectories.
