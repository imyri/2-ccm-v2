## 2. Core Specifications
- **Tool/Product**: Claude Code (Official Anthropic CLI) + Model Context Protocol (MCP).
- **Primary Function**: Terminal-based AI agent that can scan files, run commands, and post to socials via MCP bridges.
- **Free Tier Details**: Claude Code itself is free (open preview), but requires a Claude Pro ($20/mo) or API credits. 
- **Ease of Use**: 4/10 (Requires terminal knowledge and MCP server configuration).

## 3. The "Pain & Gain"
- **Pain Point**: Constant context-switching between content creation and manual scheduling on social dashboards.
- **The Gain**: "Workflow as a Code." Total delegation of the distribution loop to an agent that lives in your workspace.

## 4. Free Alternatives (Global Directive: Free Resources)
- **Gemini CLI**: Google's terminal agent. 
    - **Pros**: Generous free tier (1k requests/day), built-in Search grounding.
    - **Cons**: Newer, fewer community MCP servers than Claude.
- **Aider / OpenCode**: Open-source local agents. Can be connected to local models (Ollama) for a 100% free intelligence layer.
